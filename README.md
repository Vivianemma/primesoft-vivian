  <h4>steps</h4>
<p>step 1: Start
  <br>
Step 2: Declare variables a, b, c, D, x1, x2, rp and ip;<br>
Step 3: Calculate discriminant<br>
         D←b2-4ac<br>
Step 4: If D≥0<br>
              r1←(-b+√D)/2a<br>
              r2←(-b-√D)/2a <br>
              Display r1 and r2 as roots.<br>
        Else     <br>
              Calculate real part and imaginary part<br>
              rp←b/2a<br>
              ip←√(-D)/2a
              Display rp+j(ip) and rp-j(ip) as roots<br>
  Step 5: Stop </p>
